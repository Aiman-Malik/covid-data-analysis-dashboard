# COVID-19 Data Analysis Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Power BI dashboard analyzing COVID-19 data to identify trends, patterns, and key insights related to cases, deaths, and recovery rates.

## 🎯 Objectives
- Analyze global COVID-19 trends
- Track confirmed cases, deaths, and recoveries
- Identify affected regions and patterns
- Enable data-driven insights using visual analytics

## 📁 Dataset
- Source: Public COVID-19 dataset (Kaggle / WHO / etc.)
- Format: CSV / Excel

## 📊 Key Features
- Interactive dashboard with filters and slicers
- KPI indicators (Total Cases, Deaths, Recoveries)
- Time-based trend analysis
- Region-wise comparison
- Visual insights using charts and maps

## 📌 Key Insights
- Identified peak periods of COVID spread
- Highlighted regions with highest case counts
- Compared recovery and mortality rates
